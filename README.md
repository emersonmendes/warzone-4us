# warzone-4us


Stats page for Call of Duty Warzone.


# Dev


## Start server

```bash

cd server
npm install
export COD_CREDENTIALS='[{"user":"cod.user@mail.com", "pass":"1234"}]' && npm start

```

## Start server debug

```bash

cd server
npm install
export COD_CREDENTIALS='[{"user":"cod.user@mail.com", "pass":"1234"}]' && npm run start-debug

```

## Start client web

```bash

cd web
npm start

```
